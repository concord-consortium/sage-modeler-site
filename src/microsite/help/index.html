Work with SageModeler “documents”

Open a new SageModeler document
When you load sagemodeler.concord.org, you’re presented with two options: “Open document or browse examples” and “Create new document.” Click “Create new document” to get a blank canvas and build your own model.
Save a SageModeler document
To save a SageModeler document, go to the file menu in the upper left (three horizontal lines with an arrow) and select Save. Choose to save to Google Drive or Local File. Files will be saved with a .sagemod extension.
Open a saved SageModeler document
To open a saved document, go to the file menu in the upper left (three horizontal lines with an arrow) and select Open. Navigate to Google Drive or a Local File, then double-click the file name or select it and click “Open.”
Share my SageModeler document
To share a document, go to the file menu in the upper left (three horizontal lines with an arrow) and select Share, Get link to shared view. In the pop-up box, click the button to “Enable sharing.” Copy the link to paste into an email message, Google Doc, or other location. Or get the embed code for including in web pages. (Note: The LARA link is generally used by authors of activities in Concord Consortium’s STEM Resource Finder.)

Add variables and connections to a diagram or model

Add a variable to the canvas
To add a variable to the canvas, click the “New Image” icon in the upper left of the SageModeler tool or drag the icon onto the canvas. Search for an image using the Image Search (which searches for public domain images on openclipart.org). Once you’ve located an image, click Add Image to add it your palette. Search for additional images or close the “New Image” box. Now drag the image from the palette to the canvas.
Remove a variable from the canvas
To remove a variable from the canvas, click the variable to select it, then tap the delete key on your keyboard, or click the paintbrush icon in the menu on the right, then click “Delete.”
Add images to variables
If you have added blue boxes to represent variables and decide to use images, click a blue box to select it, then click the paintbrush icon in the menu on the right. Now click the image icon there and select “New.” Search for an image using the Image Search (which searches for public domain images on openclipart.org). Select an image and click “Add image.”
Connect variables
After you’ve added images to your canvas, connect them with links to show which variable affects the other. Select an image, click the arrow, and drag it to another image, then release.
Remove connections between variables (break a link)
To remove a connection (link) between variables, click the link to select it (the arrow will turn yellow like a highlighter), then tap the delete key on your keyboard, or click the paintbrush icon in the menu on the right, then click “Delete link.”
Add a label to a connection
To add a label to a connection (link) between variables, click the link to select it (the arrow will turn yellow like a highlighter), then click the paintbrush icon in the menu on the right. Type a title for the connection.

General Questions

What is SageModeler and how do I use it?
SageModeler is an online tool for building diagrams and system models. See About for details.
Can SageModeler work on a tablet device?
SageModeler can work on a tablet/mobile device but since it is a work in progress, not all functionality is yet available in the tablet format.
Do I need to log in?
You don’t need to log in to use SageModeler. However, if you want to save your SageModeler documents on Google Drive, you will need to create a Google account and connect your Google account to SageModeler.
How do I report bugs?
Software glitches and bugs can sometimes make their way onto web pages. If you would like to report a bug please send an email to sagemodeler@concord.org. Describe the problem with as much detail as possible, and attach the CODAP data file(s). Thank you for your help in CODAP’s development!
What are the system requirements for SageModeler?
To run SageModeler, you will need a computer with Internet connection and a relatively recent web browser. That’s it!
Will SageModeler work offline?
When an Internet connection is not always present, SageModeler will function offline if loaded online beforehand. SageModeler will only be able to accept local data from a computer in CSV or TXT format in this mode. Unfortunately embedded web pages and plugins will not be able to load.

Licensing and Permissions

Can I use SageModeler and/or screenshots of SageModeler in a textbook, article, etc.?
Sure! You can even write curriculum materials about SageModeler. You may also use the SageModeler logo when referencing SageModeler. Our work is licensed under an MIT Media License. This means you’re free to copy and share SageModeler materials. We ask that you provide attribution to the Concord Consortium and CREATE for STEM Institute at Michigan State University and the URL sagemodeler.concord.org.
How much does SageModeler cost to use? Do I need a license?
SageModeler is (and always will be) free. You don’t need a license to use SageModeler in a school, home, library, or anywhere else. The development and maintenance of SageModeler is maintained through grants.
How can I contribute to the SageModeler’s development?
If you’d like to contribute to SageModeler, email us at sagemodeler@concord.org.

SageModeler in Schools

What NGSS and Common Core Standards would using SageModeler cover?
The Next Generation Science Standards have identified eight science and engineering practices that all students should learn. Analyzing and Interpreting Data is one of the eight practices. The Common Core Mathematics Standards addressed fall under the Domains of Measurement & Data and Statistics and Probability. It would of course be dependent on the types of activities that students engaged in.
How can I design lessons that use SageModeler?
There are many ways to incorporate SageModeler into your lessons. Check out the Curriculum page for examples. You can also email us at sagemodeler@concord.org with questions.
How do I add instructions to SageModeler documents?
You can add text boxes by clicking the text box icon and/or web pages by dragging URLs directly to the SageModeler canvas then share your SageModeler document link with your students. You can also connect with the SageModeler team for specific questions. We are here to help!
How can I track students’ progress?
